# README  

## What is PostChat  
This web application gives you a experience to have chat with someone.  
You can POST, EDIT, CONFIRM, and DELETE.

## Dependency  
ruby : 2.2.2 or more  
Rails : 5.1 or more  

## Setup  
First, install *rails*  
'$ gem install rails -v 5.1.1'

Start the Rails server
'$ rails s'
